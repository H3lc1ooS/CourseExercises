Tecnologias utilizadas na programação Web

Se você já investigou assuntos relacionados a elaboração de páginas WEB, muito provavelmente ouviu falar de 3 tecnologias básicas:

  • HTML
  • CSS
  • JavaScript

Estas tecnologias, embora presentes em todas as páginas da WEB, têm causado muitas dúvidas aos iniciantes em programação, e o objectivo deste artigo é explicar o que cada uma delas é, apresentar as suas diferenças e as suas funções dentro da página.

HTML (HyperText Markup Language)

HTML, abreviação de HyperText Markup Language, é uma linguagem de marcação usada na elaboração de páginas WEB. Esta linguagem pode ser interpretada pelos navegadores.

Dentro do site, o HTML funciona como o alicerce, pois é ele o responsável por determinar o que estará presente no nosso site.

CSS (Cascading Style Sheets)

CSS, abreviação adoptada para Cascading Style Sheets, serve para adicionar estilo ao nosso documento html. O código do CSS pode ser aplicado de 3 formas:

  1. Aplicando directamente nas tags que se pretende estilizar
  2. Colocando dentro da tag STYLE
  3. Adicionando um link para o aquivo CSS que contém os estilos

JavaScript

O JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

O JavaScript tem muitas aplicações dentro do documento web, tais como:

  • Adicionar animações
  • Fazer a conexão entre o Front-End e o Back-End
